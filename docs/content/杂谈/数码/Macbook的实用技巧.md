# Macbook 的实用技巧

Macbook 拥有强大的触控板和高效的快捷键系统，掌握这些技巧能让你的工作效率倍增。

## 1. 触控板手势 (Trackpad Gestures)

Mac 的触控板体验是业界标杆，建议在 **系统设置 -> 触控板** 中开启所有手势。

*   **三指拖移**：这是最推荐开启的功能（在辅助功能中开启），选中窗口或文本后，三指轻触移动即可拖拽，手指不累。
*   **双指右键**：两根手指同时点击触控板，相当于鼠标右键。
*   **三指上推**：启动 “**调度中心**” (Mission Control)，查看所有打开的窗口和桌面。
*   **四指捏合**：打开 “**启动台**” (Launchpad)，查看所有安装的应用。
*   **四指外扩**：显示桌面，隐藏所有窗口。

## 2. 常用快捷键 (Shortcuts)

甚至可以尝试丢掉鼠标，全靠键盘和触控板。

*   `Command + Space`：呼出 **聚焦搜索** (Spotlight)，Mac 的灵魂。
*   `Command + C/V`：复制/粘贴。
*   `Command + Option + V`：剪切（Mac 上没有直接的 Ctrl+X 剪切文件，而是先复制，再用这个快捷键移动）。
*   `Command + W`：关闭当前标签页或窗口。
*   `Command + Q`：完全退出当前程序。
*   `Command + Tab`：在打开的应用程序之间切换。
*   `Command + ,`：打开当前应用的偏好设置（绝大多数 App 通用）。

## 3. 聚焦搜索 (Spotlight)

`Command + Space` 不仅仅是找文件，它更是全能工具箱：

*   **启动应用**：输入应用名首字母即可快速打开 App。
*   **计算器**：直接输入算式，如 `1024 * 768`，直接显示结果。
*   **汇率换算**：输入 `100 USD`，直接显示人民币金额。
*   **单位换算**：输入 `100 kg` 等。

## 4. 截图与录屏

Mac 自带的截图非常强大，不再需要第三方工具。

*   `Command + Shift + 3`：全屏截图。
*   `Command + Shift + 4`：截取选定区域。
*   `Command + Shift + 4` 后按 `Space`（空格键）：截取指定窗口（带阴影效果，非常美观）。
*   `Command + Shift + 5`：调出录屏和截图工具栏，可以录制屏幕视频。

## 5. 触发角 (Hot Corners)

将鼠标移动到屏幕的四个角落，可以触发特定功能。
设置位置：**系统设置 -> 桌面与程序坞 -> 触发角**。

推荐设置：
*   右下角：**快速备忘录**（随手记灵感）。
*   左下角：**锁定屏幕**（离开工位时顺手一划）。

## 6. 预览 (Quick Look)

选中任何文件（图片、文档、视频、PSD 等），按下 **空格键 (Space)**，即可快速预览内容，无需打开相应的软件。
