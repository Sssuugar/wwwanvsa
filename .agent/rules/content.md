**重要提示**：在书写中文或英文时，如果涉及到引号内的加粗文本，必须确保与静态网站生成器（如 VitePress）的兼容性。
1.  **引号内加粗**：当需要加粗被引号（直引号 `"` 或弯引号 `“ ”`）包裹的短语时，**务必**将加粗符号 (`**`) 放在引号**内部**。
    *   ❌ **错误写法**：`**“核心概念”**` 或 `**"Key Concept"**`
    *   ✅ **正确写法**：`“**核心概念**”` 或 `“**Key Concept**”`
    *   **原因**：将加粗符号放在引号外部会导致许多 Markdown 解析器（如 VitePress）渲染失败，直接显示星号而无法加粗。
2.  **禁止“自作聪明”地移动引号**：不要试图将引号也包含在加粗符号内（例如 `**“核心概念”**` -> `**“核心概念”**` 虽然能渲染但语义不同），除非用户明确要求。我们的目标是仅加粗引号内的**文本内容**，而不包括标点符号。
